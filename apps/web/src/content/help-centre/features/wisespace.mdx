# Wisespace: Virtual Classroom

Wisespace is Tutorwise's built-in virtual classroom - your complete online tutoring environment with whiteboard, screen sharing, file uploads, and session recording.

<CalloutBox type="info">
**Integrated Solution:** No need for Zoom or Google Meet. Wisespace is built directly into Tutorwise for seamless online sessions.
</CalloutBox>

## What Is Wisespace?

**All-in-One Virtual Classroom:**
- Interactive whiteboard
- Screen sharing (tutor & student)
- File uploads and sharing
- Session recording (automatic)
- Real-time collaboration
- Built-in video/audio chat

**Why Use Wisespace?**
- Integrated with bookings (one-click join)
- Automatic session recording for review
- Boosts your Credibility Score (+3 points for 80%+ usage)
- Professional tutoring environment
- No third-party apps needed

<CalloutBox type="success">
**CaaS Boost:** Using Wisespace for 80%+ of your sessions earns +3 Credibility Score points in the Digital Professionalism bucket!
</CalloutBox>

## Joining a Wisespace Session

### For Students

**Step 1: Find Your Session Link**
- Check booking confirmation email
- Go to [Dashboard](/dashboard) → Upcoming Sessions
- Click "Join Wisespace" button 5 minutes before start

**Step 2: Join the Room**
1. Click "Join Session"
2. Allow camera/microphone permissions
3. Test audio and video
4. Click "Enter Classroom"

**Step 3: During the Session**
- Tutor's video appears top-right
- Whiteboard in center
- Tools on left sidebar
- Chat on right panel

### For Tutors

**Step 1: Prepare Your Space**
- Test camera and mic 10 minutes early
- Prepare materials to share
- Have whiteboard templates ready
- Ensure stable internet

**Step 2: Start the Session**
1. Go to [Bookings](/bookings)
2. Find upcoming session
3. Click "Start Wisespace"
4. Session room opens automatically

**Step 3: Admit Student**
- Student appears in waiting room
- Click "Admit" to let them in
- Session officially begins
- Recording starts automatically

<CalloutBox type="tip">
**Pro Tip:** Start 5 minutes early to troubleshoot any technical issues before the student joins!
</CalloutBox>

## Wisespace Tools & Features

### Interactive Whiteboard

**Drawing Tools:**
- Pen (multiple colors and sizes)
- Highlighter
- Eraser
- Shapes (circle, square, arrow)
- Text tool
- Sticky notes

**Collaboration:**
- Both tutor and student can draw
- Multi-page whiteboard (add new pages)
- Undo/redo history
- Clear board option
- Save whiteboard as PDF

**Use Cases:**
- Solving math problems step-by-step
- Diagramming concepts
- Annotating text or images
- Brainstorming together
- Visual explanations

### Screen Sharing

**What You Can Share:**
- Entire screen
- Specific application window
- Browser tab only
- PDF or document

**Controls:**
- Start/stop sharing anytime
- Switch shared content
- Student can request control
- Tutor approves control requests

**Best For:**
- Showing presentations
- Walking through documents
- Demonstrating software
- Sharing educational videos
- Reviewing student work

### File Sharing

**Upload During Session:**
- PDFs, images, documents
- Maximum 10MB per file
- Drag and drop support
- Files saved in session history

**File Actions:**
- Download files
- View in classroom
- Annotate PDFs on whiteboard
- Share screen to present

### Session Recording

**Automatic Recording:**
- Starts when session begins
- Records video, audio, and whiteboard
- Saves automatically when session ends
- Available for 90 days

**Access Recordings:**
1. Go to [Bookings](/bookings)
2. Find completed session
3. Click "View Recording"
4. Watch or download

**Privacy:**
- Both parties notified of recording
- Can't disable (platform policy for protection)
- Only participants can access
- Auto-deleted after 90 days

<CalloutBox type="warning">
**Privacy Notice:** All sessions are recorded for quality assurance and dispute resolution. Both parties are notified at session start.
</CalloutBox>

## Technical Requirements

**Minimum Requirements:**
- Chrome, Firefox, or Safari browser (latest version)
- 5 Mbps internet speed (10 Mbps+ recommended)
- Webcam and microphone
- Desktop or laptop (mobile support limited)

**Recommended Setup:**
- 20 Mbps+ internet
- HD webcam
- Headset with microphone (reduces echo)
- External tablet for whiteboard (optional)
- Dual monitors (one for Wisespace, one for materials)

**Troubleshooting Connection:**
- Close unnecessary browser tabs
- Disable VPN if having issues
- Use wired connection if possible
- Test connection at speedtest.net

## Best Practices

**For Tutors:**
✅ Test your setup before each session
✅ Use headset to prevent echo
✅ Position camera at eye level
✅ Good lighting (face camera toward window)
✅ Clean, professional background
✅ Prepare whiteboard templates in advance
✅ Have backup materials ready

**For Students:**
✅ Join 2-3 minutes early
✅ Find quiet space with minimal distractions
✅ Have notebook and materials ready
✅ Test camera/mic before session
✅ Use headphones for better audio
✅ Ask questions during session

## Troubleshooting

**Q: I can't hear the student. What do I do?**
A: Check:
- Your volume settings
- Student's mic isn't muted
- Browser permissions allowed
- Try refreshing the page

**Q: Whiteboard is laggy. How to fix?**
A: Try:
- Close other browser tabs
- Reduce video quality in settings
- Use wired internet connection
- Clear browser cache

**Q: Session recording failed. Where is it?**
A: If recording fails:
- Check your booking page after 24 hours
- Contact support with session ID
- We may be able to recover it
- Prevent future issues by testing connection

**Q: Can I use Wisespace on mobile/tablet?**
A: Limited support currently. Works best on desktop/laptop with Chrome browser. Mobile app coming soon!

**Q: How do I share my screen on Mac?**
A: You may need to:
- Go to System Preferences → Security & Privacy
- Allow browser to record screen
- Restart browser
- Try screen share again
